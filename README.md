# 🚀 Talent Tuner

> **An AI-Powered Developer Intelligence & Career Acceleration Ecosystem**

Talent Tuner is not just a GitHub analyzer or a resume checker.
It is a full-stack AI-powered intelligence ecosystem designed to bridge the gap between **student capabilities** and **industry expectations**.

Instead of relying on self-claimed resume buzzwords, Talent Tuner extracts **verified technical intelligence** directly from real GitHub repositories and transforms them into recruiter-consumable developer intelligence profiles.

---

# 🌌 Vision

Modern hiring is broken.

Students:

* Learn random technologies without direction
* Build projects that never get visibility

* Depend on weak resumes that fail to reflect real technical depth

Recruiters:

* Waste time filtering resumes manually
* Cannot verify real developer capabilities
* Rely on keyword matching instead of technical proof

✅ Analyzing real repositories

✅ Mapping developers against market requirements

✅ Generating deterministic compatibility scoring
✅ Guiding developers toward industry readiness

# ⚡ Installation & Setup

# 1️⃣ Clone Repository

```bash
git clone <your-repo-link>
cd talent-tuner
```

---

# 2️⃣ Backend Setup

```bash
cd backend
npm install
```

Create `.env`

```env
DATABASE_URL=your_neon_database_url
JWT_SECRET=your_secret
OPENAI_API_KEY=your_openai_key
EMAIL_USER=your_email
EMAIL_PASS=your_password
```

Run Prisma:

```bash
npx prisma generate
npx prisma db push
```

Start backend:

```bash
node server.js
```

---

# 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
